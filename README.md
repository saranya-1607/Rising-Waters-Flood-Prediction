# 🌊 Rising Waters - AI Flood Prediction System

## 📌 Project Overview

Rising Waters is an AI-powered Flood Prediction System developed using Machine Learning and Flask. The application predicts flood risk by analyzing environmental and geographical factors such as rainfall intensity, drainage systems, urbanization, watershed conditions, climate change, and other contributing parameters.

The system provides instant flood risk predictions along with safety recommendations through an interactive web interface.

---

## 🚀 Features

* 🌧 Predicts Flood Risk using Machine Learning
* 🤖 XGBoost based prediction model
* 📊 Probability-based risk estimation
* 💡 Flood safety recommendations
* 🌧 Animated rainfall background
* 📱 Responsive modern UI
* 🔄 Reset functionality
* ⚡ Fast Flask backend

## ⚙ Installation

Clone the repository

```
git clone https://github.com/BASITH2602/Rising-Waters-Flood-Prediction.git
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
python app.py
```

Open

```
http://127.0.0.1:5000
```

## 📈 Machine Learning Model

Models experimented:

* Decision Tree
* Random Forest
* KNN
* XGBoost

The XGBoost model achieved the best performance and was selected for deployment.
